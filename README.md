## Deskripsi Project

Project ini bertujuan untuk melakukan:
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Interactive data visualization dashboard

terhadap dataset penjualan video game global untuk menemukan insight terkait:
- tren penjualan game
- genre paling populer
- platform paling sukses
- preferensi regional terhadap genre game
- dan lainnya

Dataset yang digunakan: https://www.kaggle.com/datasets/gregorut/videogamesales
